# Privacy Policy for GitHub Tools Plugin

**Effective Date:** 2025-06-27

## Data Collection

The GitHub Tools Plugin ("the Plugin") does **not** collect, store, or transmit any personal data to third parties beyond what is necessary to fulfill its core functionality.

### What the Plugin Accesses

1. **GitHub Public Data** — When you use the Plugin's tools (search repositories, view README, list issues, get repository info), it fetches **publicly available data** from the GitHub REST API. These requests are made on your behalf using the credentials you provide.

2. **GitHub Token (Optional)** — If you configure a GitHub Personal Access Token, it is stored securely within the Dify platform and used solely to authenticate API requests to GitHub. The token is never transmitted to any third party or logged.

3. **No User Data Collection** — The Plugin does not:
   - Collect analytics or usage statistics
   - Use cookies or tracking mechanisms
   - Store any personal information
   - Share data with any third-party services

## Data Processing

All API requests are made directly from the Dify runtime environment to the GitHub API. No intermediate servers or proxies are involved.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the Plugin's repository.

## Contact

For questions about this privacy policy, please create an issue in the Plugin's GitHub repository.
